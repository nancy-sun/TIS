# TIS

## Description

This is a full-stack hackathon project @ Brainstaion, winning the first place.  
TIS generates travel destinations based on user's mood/vibe, with external expedia links to help users get traveling packages. 
  
## Tech Stack  
- React  
- Sass
- Node.js  
- Express
